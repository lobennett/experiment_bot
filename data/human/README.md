# Human Reference Data

Trial-level behavioral data from Eisenberg et al. (2019) Self Regulation Battery.

## Source

Download `stop_signal.csv.gz` and `stroop.csv.gz` from:
https://github.com/IanEisenberg/Self_Regulation_Ontology/tree/master/Data/Complete_02-16-2019/Individual_Measures

Place the uncompressed CSVs in this directory as `stop_signal.csv` and `stroop.csv`.

## Reference

Eisenberg, I. W., Bissett, P. G., Enkavi, A. Z., Li, J., MacKinnon, D. P., Marsch, L. A., & Poldrack, R. A. (2019). Uncovering the structure of self-regulation through data-driven ontology discovery. *Nature Communications*, 10(1), 2319.

## Archived Data

The `archive_rdoc/` directory contains the previously used RDoC behavioral battery session-level data.
